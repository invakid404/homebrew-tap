# Invakid404 Tap

## How do I install these formulae?

`brew install invakid404/tap/<formula>`

Or `brew tap invakid404/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "invakid404/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
